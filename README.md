# AntiHile Project

A TypeScript-based anti-cheat system.

## Project Setup

1. Install Node.js from https://nodejs.org/
2. Install project dependencies:
   ```
   npm install
   ```

## Development

To start development:
```
npm run dev
```

## Build

To build the project:
```
npm run build
```

## Run

To run the built project:
```
npm start
```
